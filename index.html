About ref

Ref is a powerful lightweight command-line reference manager. It is built on python and vim. Screenshot.

Installation

Under linux/unix/Mac(?):

$ git clone https://github.com/jzbontar/ref.git
$ cd ref
$ sudo python setup.py

Later we might add pip install or something fancy like that.
